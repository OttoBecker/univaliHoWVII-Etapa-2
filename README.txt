Passo a passo para a execução do sistema da etapa 2:

1 - Instale o NodeJs: https://nodejs.org/pt;

2 - Clone o repositório;

3 - Ajuste os dados do banco de dados, como o user, password e database (De preferência com o nome "univali"), no arquivo db.js dentro da pasta src;

4 - Utilize o comando "npm install" no terminal para instalar as dependências necessárias;

5 - Utilize o comando "npm init -y" no terminal caso não possua o arquivo "package-lock.json";

6- Utilize o comando "npm run iniciarProjeto" para inicar o projeto;
